# 47 Industries

**Building software, hardware, and products that move things forward.**

47 Industries is a technology company focused on 3D printing, custom manufacturing, and software development. We build our own products and help others build theirs.

---

## What We Build

| Product | Description | Stack |
|---|---|---|
| [47industries.com](https://47industries.com) | E-commerce platform — 3D printing, custom manufacturing, web & app dev services | Next.js, Prisma, MySQL, Railway |
| [MotoRev](https://motorevapp.com) | iOS motorcycle tracking app — GPS, social, safety, garage management | Swift, Node.js |
| [BookFade](https://github.com/47-Industries/bookfade) | Smart barber booking platform | React Native, Expo |
| [LeadSlicer](https://github.com/47-Industries/LeadSlicer) | Lead generation tool | — |

## Services

- **3D Printing** — Mass-produced items and custom print orders
- **Web Development** — Full-stack web applications
- **App Development** — iOS and cross-platform mobile apps
